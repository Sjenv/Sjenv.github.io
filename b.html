Organic Waste Management Division:-


Organic Waste Convertor (OWC) is a bio-mechanical Composting System which consists of the OWC machine, Curing System and a number of optional accessories for specific waste categories.


OWC is an easy to use Decentralized Waste Management System to turn large amounts of organic waste such as household kitchen / hotel kitchen wastes, garden waste, food processing waste etc. into compost. The system is designed to eliminate odour and also to remove the problem of irritants such as flies and rats. We have tied up with presumed constructers for these OWC machines.
